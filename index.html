Syntactically Awesome Style Sheets

By: Sahil H. Mobaidin
What is SASS? Sass is a preprocessor scripting language that is interpreted or compiled into Cascading Style Sheets (CSS). SassScript is the scripting language itself. https://en.wikipedia.org/wiki/Sass_(stylesheet_language)
Why use SASS?
  • Intelligent
  • Reusable
  • Maintainable
  • Fun to use
Nesting 
    
    #main p
  color: #00ff00
  width: 97%
  .redbox 
    background-color: #ff0000
    color: #000000
    
  
      
      #main p {
  color: #00ff00;
  width: 97%; 
}
#main p .redbox {
  background-color: #ff0000;
  color: #000000; 
}
Nesting :pseudo classes 
    
    #main 
  color: black;
  .link 
    font-weight: bold
    &:hover
      color: red
    
  
      
      #main {
  color: black; 
}

#main .link {
  font-weight: bold; 
}
#main .link:hover {
  color: red; 
}
Also applies for other pseudo classes like:
::before, ::after, :focus...etc
Variables 
    
    $mainColor: #3bbfce
$margin: 16px
      
.content_navigation 
  border-color: $mainColor
                
.border 
  padding: $margin / 2
  margin: $margin / 2
  border-color: $mainColor
  
    
    .content_navigation {
  border-color: #3bbfce;
}
                
.border {
  padding: 8px;
  margin: 8px;
  border-color: #3bbfce;
}
Mixins 
	
	=transform($property)
  -webkit-transform: $property
  -ms-transform: $property
  transform: $property
	
.box
  +transform(rotate(30deg))

.title
  +transform(rotate(60deg))	

	
	.box {
  -webkit-transform: rotate(30deg);
  -ms-transform: rotate(30deg);
  transform: rotate(30deg);
}

.title {
  -webkit-transform: rotate(60deg);
  -ms-transform: rotate(60deg);
  transform: rotate(60deg);
}
Operators 
	
	.container
  width: 100%
	
.article
  float: left
  width: 600px / 3px

.aside
  float: right
  width: 300px / 10px

	
	.container {
  width: 100%;
}

.article {
  float: left;
  width: 200px;
}

.aside {
  float: right;
  width: 30px;
}
Also applies for other operators: + - * / and %

Lists 
	
	$buttonConfig: 'save' 50px, 'cancel' 70px   

@each $tuple in $buttonConfig 
  .button-#{nth($tuple, 1)} 
    width: nth($tuple, 2)
	

	
	.button-save
  width: 50px;

.button-cancel
  width: 70px;
While 
	
	$i: 6;
@while $i > 0
  .item-#{$i} 
    width: 2em * $i; 
  $i: $i - 2;
	

	
	.item-6 {
width: 12em; }

.item-4 {
	width: 8em; 
}

.item-2 {
  width: 4em; 
}
For 
	
	@for $i from 1 through 3 
  .item-#{$i} 
    width: 2em * $i;
	

	
	.item-1 {
  width: 2em; 
}

.item-2 {
  width: 4em; 
}

.item-3 {
  width: 6em; 
}
