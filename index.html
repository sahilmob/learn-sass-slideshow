Syntactically Awesome Style Sheets

By: Sahil H. Mobaidin
What is SASS? Sass is a preprocessor scripting language that is interpreted or compiled into Cascading Style Sheets (CSS). SassScript is the scripting language itself. https://en.wikipedia.org/wiki/Sass_(stylesheet_language)
Why use SASS?
  • Intelligent
  • Reusable
  • Maintainable
  • Fun to use
Nesting 
    
    #main p
  color: #00ff00
  width: 97%
  .redbox 
    background-color: #ff0000
    color: #000000
    
  
      
      #main p {
  color: #00ff00;
  width: 97%; 
}
#main p .redbox {
  background-color: #ff0000;
  color: #000000; 
}
Nesting :pseudo classes 
    
    #main 
  color: black;
  .link 
    font-weight: bold
    &:hover
      color: red
    
  
      
      #main {
  color: black; 
}

#main .link {
  font-weight: bold; 
}
#main .link:hover {
  color: red; 
}
Also applies for other pseudo classes like:
::before, ::after, :focus...etc
Variables 
    
    $mainColor: #3bbfce
$margin: 16px
      
.content_navigation 
  border-color: $mainColor
                
.border 
  padding: $margin / 2
  margin: $margin / 2
  border-color: $mainColor
  
    
    .content_navigation {
  border-color: #3bbfce;
}
                
.border {
  padding: 8px;
  margin: 8px;
  border-color: #3bbfce;
}
